Controls

Up - W, Up Arrow, or
Left - A, Left Arrow, or
Right - D, Right Arrow, or
Down - S, Down Arrow, or
Light Mode - Light mode can be switched to with the -/_ key or button
Dark Mode - Dark mode is the default setting but if you switch to light mode, you can go back to it with the +/= key or button
Faces - The player can have 3 different faces:
  • Normal - The 1 key
  • Eyes Only - The 2 key
  • No Face - The 3 key

How To Play

Using the controls mentioned above you can move around, by running into the box you push it. The goal of the game is to move the box to the goal (the black and white checkerboard) which adds 1 to your score. see how high of a score you can get!
